## 🛡️ 关于人机验证打码平台的配置

可解决游戏签到时遇到人机验证导致无法完成签到的问题，打码平台是第三方平台，可自动完成人机验证，一般需要付费。

### 相关插件配置

- `geetest_url` - 打码平台接口 URL 地址
- `geetest_params` - 打码平台接口 URL 参数
- `geetest_json` - 打码平台接口请求发出的 JSON 模板
